# Movie Finder.

Checkout the Top IMDb rated movies, also search for the one you would like to know more about.

![](https://github.com/gerardogaravito/movie-finder/blob/main/public/movie_finder.png)

This Project is build using:

- NextsJS as a React framework
- ReactQuery to request data and control the serverStore
- StyledComponents for styling, letting write CSS in TypeScript
- TypeScript

API by IMDb(https://imdb-api.com/api)

- Hosted by Vercel
